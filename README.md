# FitNotes Analytics

This project enriches raw FitNotes workout logs by adding structured classifications such as muscle groups, movement patterns, and load type (external vs. bodyweight). With this added context, it enables analysis of training volume, progression, frequency, balance across muscle groups, and longer-term performance trends.

## Getting Started

```
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
